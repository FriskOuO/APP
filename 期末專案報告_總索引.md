## 📊 專案統計

### 程式碼規模
- **總代碼行數**: ~3,000+ 行
- **組件數量**: 15+ 個
- **狀態數量**: 15+ 個
- **API 端點**: 3 個

### 技術棧
- **前端**: React 19, XState 5, Tailwind CSS

## 🔧 開發環境設置

### 前置需求
```bash
Node.js >= 14.0.0
npm >= 6.0.0
```

### 安裝步驟
```bash
# 1. 克隆專案
git clone <repository-url>
cd APP_FINAL

# 2. 安裝依賴
npm install

# 3. 設置環境變數
cp .env.example .env
# 編輯 .env 填入 Gmail 設定

# 4. 啟動後端服務
cd server
node server.js

# 5. 啟動前端 (新終端)
npm start
```

### Web 版本
```bash
npm start
# 開啟 http://localhost:3000
```

### 移動端版本
```bash
npm run android  # Android
npm run ios      # iOS
```
